========================
Importing TEI XML files
========================

One of the most important corpus for Estonian is `Tasakaalus Korpus`_, which
can be downloaded as XML files.

Estnltk has :class:`estnltk.teicorpus.parse_tei_corpus` function for reading a single XML file
and :class:`estnltk.teicorpus.parse_tei_corpora` to read a full directory of XML data.

.. _Tasakaalus Korpus: http://www.cl.ut.ee/korpused/
