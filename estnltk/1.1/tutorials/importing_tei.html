Importing TEI XML filesÂ¶

One of the most important corpus for Estonian is Tasakaalus Korpus, which can be downloaded as XML files.

Estnltk has estnltk.teicorpus.parse_tei_corpus function for reading a single XML file and estnltk.teicorpus.parse_tei_corpora to read a full directory of XML data.
